# 🏦 Bank System - OOP Project (C++)

---

## 🧠 About the Project | عن المشروع

**English:**  
This project is a simple **Bank Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles.  
It demonstrates how OOP concepts like classes, inheritance, encapsulation, and polymorphism can be applied to build a real-world banking system simulation.  

**العربية:**  
مشروع **Bank System** هو نظام بنكي بسيط مكتوب بلغة **++C** باستخدام مفاهيم **البرمجة الكائنية التوجه (OOP)**.  
بيوضح كيف فينا نطبّق مفاهيم الـ OOP مثل الوراثة، التغليف، التعدد الشكلي، وإنشاء الأصناف لبناء نظام بنكي حقيقي بشكل مبسط.

---

## ⚙️ Features | الميزات

**English:**
- 👤 Manage clients (Add, Edit, Delete)
- 💰 Manage bank accounts
- 🏦 Perform transactions:
  - Deposit money  
  - Withdraw money  
  - Transfer funds between accounts  
- 📋 Display customer and account details  
- 🔒 Simple access control (Admin / User)
- 💾 Data persistence using text files  

**العربية:**
- 👤 إدارة العملاء (إضافة، تعديل، حذف)  
- 💰 إدارة الحسابات البنكية لكل عميل  
- 🏦 تنفيذ العمليات البنكية:
  - الإيداع  
  - السحب  
  - التحويل بين الحسابات  
- 📋 عرض بيانات العملاء والحسابات بشكل منظم  
- 🔒 نظام صلاحيات بسيط (مدير / مستخدم)  
- 💾 حفظ البيانات داخل ملفات نصية (TXT Files)

## 🧩 OOP Concepts Used | مفاهيم البرمجة الشيئية المطبقة

**English:**
- **Encapsulation:** Organizing data and hiding internal implementation  
- **Inheritance:** Creating subclasses that extend base classes  
- **Polymorphism:** Overriding methods to provide different behaviors  
- **Abstraction:** Hiding complex logic and exposing only essential functions  

**العربية:**
- **التغليف (Encapsulation):** تنظيم الكود ضمن كائنات ومنع الوصول المباشر للبيانات  
- **الوراثة (Inheritance):** إنشاء أصناف فرعية ترث خصائص الأصناف الأساسية  
- **تعدد الأشكال (Polymorphism):** تغيير سلوك الدوال في الأصناف المشتقة  
- **التجريد (Abstraction):** إخفاء التفاصيل المعقدة وإظهار الوظائف الضرورية فقط
